## 🔐 **阶段三：云同步（第5-8周）**

### 目标
- 实现多设备数据同步
- 为付费功能铺路
- 大幅提升用户粘性

### 技术选型

**推荐：Firebase（最快上手）**
- 免费配额充足（100 并发用户/月）
- 自带认证系统
- 实时数据库 + Firestore 选择
- 学习成本低

**备选：Supabase（更灵活）**
- PostgreSQL + 实时 API
- 免费 500MB 存储
- 需要一定后端知识

### 任务列表

#### 3.1 Firebase 集成（5天）
- [ ] 创建 Firebase 项目
- [ ] 配置认证方式（Email、Google、GitHub）
- [ ] 设置 Firestore 数据库规则
- [ ] 安装 Firebase SDK
- [ ] 创建 `composables/useAuth.js` 认证模块
- [ ] 创建 `composables/useCloudSync.js` 同步模块

**核心代码：**
```javascript
// composables/useCloudSync.js
import { initializeApp } from 'firebase/app'
import { getFirestore, doc, setDoc, onSnapshot } from 'firebase/firestore'

export function useCloudSync() {
  const db = getFirestore()
  
  const syncToCloud = async (userId, data) => {
    await setDoc(doc(db, 'users', userId), {
      pomodoroRecords: data.pomodoroRecords,
      settings: data.settings,
      updatedAt: new Date()
    })
  }
  
  const listenToChanges = (userId, callback) => {
    return onSnapshot(doc(db, 'users', userId), (doc) => {
      callback(doc.data())
    })
  }
  
  return { syncToCloud, listenToChanges }
}
```
